# 🛺 جايك — Jayak

تطبيق توصيل ونقل بالتكتك — **نسخة تجريبية (MVP)**

الزبون يقدر يطلب تكتك يوصّله، أو يطلب توصيل بضاعة من أي محل. يوصل إشعار لكل السواقين المتاحين، وأول سواق يقبل ياخذ الطلب مع تتبع لحظي على الخريطة.

---

## الخصائص

### تطبيق الزبون
- ✅ تسجيل دخول (ينحفظ الاسم والرقم)
- ✅ قائمة جانبية (حساب، رصيد، رحلات، مواقع مختارة، تخفيضات، دعوة أصدقاء، إعدادات، عن جايك، دعم)
- ✅ **نوعين طلبات**:
  - 🛺 **رحلة عادية** — نقطة انطلاق + نقطة وصول
  - 🛒 **توصيل بضاعة** — محل + وصف البضاعة + عنوان التوصيل
- ✅ بحث عن الأماكن بالاسم + تحديد على الخريطة
- ✅ أجرة تقديرية قبل الطلب
- ✅ تتبع لحظي للسواق + خط السير + وقت الوصول

### تطبيق السواق
- ✅ تسجيل دخول (اسم + رقم + وصف التكتك)
- ✅ استقبال الطلبات (رحلة أو توصيل) مع تمييز واضح بينهن
- ✅ عرض تفاصيل التوصيل (المحل + وصف البضاعة)
- ✅ كشف حساب: إجمالي الأرباح، أرباح اليوم، عدد الطلبات، الكيلومترات، وقائمة الطلبات

## التقنيات

| المكون | التقنية |
|--------|---------|
| السيرفر | Node.js + Express |
| التتبع اللحظي | WebSocket (ws) |
| الخريطة | Leaflet + OpenStreetMap (مجاني) |
| البحث | Nominatim (مجاني) |
| التخزين | في الذاكرة (تجريبي) — قابل للترقية لـ PostgreSQL |

## التشغيل

```bash
npm install
npm start
```

- الصفحة الرئيسية: `http://localhost:3000`
- واجهة الزبون: `http://localhost:3000/ride`
- واجهة السواق: `http://localhost:3000/driver`
