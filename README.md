# 🛺 تكتك المسيب — Tuktuk Al-Musayyib

تطبيق حجز تكتك للمسيب — **نسخة تجريبية (MVP)**

الزبون يحجز تكتك، يوصل إشعار لكل السواقين المتاحين مع موقع الزبون. أول سواق يقبل ياخذ الطلب، وتظهر للزبون معلومات السواق وخط سيره ووقت وصوله مع تتبع لحظي على الخريطة.

---

## الخصائص

- ✅ حجز تكتك بتحديد الموقع على الخريطة
- ✅ إشعار فوري لكل السواقين المتاحين
- ✅ أول سواق يقبل ياخذ الطلب (منع الحجز المزدوج)
- ✅ عرض اسم السواق، رقمه، وصف تكتكه
- ✅ تتبع لحظي — موقع السواق يتحرك على الخريطة (WebSocket)
- ✅ رسم خط السير من السواق للزبون
- ✅ حساب وقت الوصول التقريبي
- ✅ أزرار اتصال مباشر بين الزبون والسواق
- ✅ إشعار الوصول وإنهاء الرحلة
- ✅ تطبيق واحد يخدم الزبون والسواق

## التقنيات

| المكون | التقنية |
|--------|---------|
| السيرفر | Node.js + Express |
| التتبع اللحظي | WebSocket (ws) |
| الخريطة | Leaflet + OpenStreetMap (مجاني) |
| التخزين | في الذاكرة (تجريبي) — قابل للترقية لـ PostgreSQL |

> **ملاحظة:** النسخة التجريبية تستخدم خريطة OpenStreetMap المجانية. نسخة الإنتاج تنتقل لـ Google Maps للدقة الأعلى، وتضيف إشعارات FCM، وقاعدة بيانات PostgreSQL.

## التشغيل محلياً

```bash
npm install
npm start
```

ثم افتح المتصفح:
- الصفحة الرئيسية: `http://localhost:3000`
- واجهة الزبون: `http://localhost:3000/ride`
- واجهة السواق: `http://localhost:3000/driver`

## التجربة

1. افتح واجهة السواق `/driver` بجهاز (أو تبويب)، سجّل بياناتك، وشغّل مفتاح "متصل"
2. افتح واجهة الزبون `/ride` بجهاز ثاني (أو تبويب)، حدد موقعك، واحجز
3. راح يوصل الطلب للسواق فوراً — اقبله وشوف التتبع

## المسارات (API)

| المسار | الوظيفة |
|--------|---------|
| `POST /api/book` | الزبون يحجز |
| `POST /api/accept` | السواق يقبل |
| `POST /api/arrived` | السواق وصل |
| `POST /api/complete` | إنهاء الرحلة |
| `POST /api/cancel` | إلغاء |
| `GET /api/stats` | إحصائيات |
